# Git Commands

1. Initialize git in repository

```bash
git init
```

2. Add files to stage

```bash
git add <file-name>
git add .
```

3. Commit the changes

```bash
git commit -m "<message>>"
```

4. Lists the commit history

```bash
git log
git log --oneline
git log --author="<user-name>"
```
